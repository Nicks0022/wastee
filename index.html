Perguntas frequentes


Como posso vender meus dispositivos usados?
Para vender dispositivos usados, basta se cadastrar na plataforma, criar um anúncio e seguir as instruções para completar a venda.
Qual é o custo para se cadastrar como vendedor?
Não há custo para se cadastrar como vendedor. Você só paga uma pequena taxa sobre cada venda realizada.
Como funciona o sistema de avaliações?
Os compradores podem avaliar os vendedores após a compra, ajudando a construir uma reputação dentro da plataforma.
O que é o plano premium?
O plano premium oferece benefícios adicionais, como maior visibilidade dos seus anúncios e taxas reduzidas em cada venda.
Como funcionam os cupons?
Os cupons permitem descontos nas compras e podem ser aplicados durante o checkout.
